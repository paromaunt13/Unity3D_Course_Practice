игра состоит из 3 игровых локаций:
1. основная локация с одним NPC, с которым можно взаимодействовать посредством диалога
этот NPC даёт 1-2 квеста, выдаёт награду по их завершении
2. локация для сбора различных ресурсов
в этой зоне также есть нейтральные враги, которых нужно убить, чтобы получить необходимый ресурс
3. локация с боссом, завершающая 
для прохождения этой локации нужно убить босса

основные компоненты в игре:
1. главное меню/меню паузы, экран game over/game completed
2. UI, отображающий количество здоровья, собранных ресурсов, диалоги, текущий квест
3. различные SFX, VFX, базовая анимация персонажей, врагов
